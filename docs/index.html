VERBA v1.0 — THE NATURAL LANGUAGE

The Language of Humans.

Verba is an interpreted, English-like programming language designed to be readable by anyone, even without a programming background.

hello.vrb 
/- A clean welcome
ask the user "What is your name?" and save to name.
say "Hello, {name}! Welcome to Verba.".
100% Readable
0.1ms Fast Execution
20+ Core Modules

Installation

Verba requires Python 3.10 or newer.

# Install locally
pip install -e .

# Run a script
verba run project.vrb

CLI Commands

The verba tool provides everything you need to manage your lifecycle.

CommandEffect
verba runExecute code.
verba checkSyntax validation.
verba replLive console.
verba formatCode cleanup.

General Syntax

Verba is designed to feel like instructions to a human.

/- Use a period to end statements.
say "Rule 1".
/- Use a colon to start blocks.
if true:
    say "Rule 2".
end.

Variables & Types

score = 100.
name = "Alice".
is_admin = true.
null_vals = null.

Arithmetic

a = 10 + 5.
b = a * 2.
c = 10 % 3.
d = 2 ** 8.

Input & Output

say "Hello!".
ask the user "Name?" and save to name.

Conditions

if x > 10:
    say "Big".
else:
    say "Small".
end.

Loops

i = 1.
while i <= 5:
    say i.
    i += 1.
end.
repeat 5 times:
    say "Again".
end.
for item in list:
    say item.
end.

Pattern Matching

match val:
    when 1: say "One".
    else: say "Else".
end.

Lists & Collections

nums = a list of 1, 2, 3.
add 4 to nums.
remove 2 from nums.

Maps (Dictionaries)

user = {"id": 1, "name": "Alice"}.
say user.name.

Functions

define add needing a, b:
    give a + b.
end.

res = the result of running add with 1, 2.

Generators

define counter:
    yield 1.
    yield 2.
end.

Classes & Objects

class Pet:
    define bark: say "Woof". end.
end.

dog = new Pet.
run dog.bark.

Pointers & Refs

x = 10.
ptr = &x.
deref ptr = 20.

Async / Await

async define job: say "Doing". end.
task = async run job.
await res = task.

Error Handling

try:
    raise "Error!".
on error saving to e:
    say e.
end.
